# Como rodar localmente o projeto 
1. selecione um diretório local, o mesmo será responsável por receber o arquivo do projeto
2. Abra sua ferramenta git e use o comando git clone https://github.com/Arthur-155/desafio-flugo.git
3. com o git aberto escreva cd desafio-flugo/ e em seguida code .
4. com o vscode aberto abra o terminal no menu superior ou pressione Ctrl + Shift + '
5. No terminal use o comando npm i ou npm install
6. Por fim, rode o comando npm run dev
